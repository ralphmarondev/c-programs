# C Programming Practice Repository

Welcome! 👋

This repository contains C programming examples, exercises, and solutions designed to help students understand core programming concepts step by step.

## 📚 Purpose

This repository is intended to:

* Reinforce lessons discussed in class
* Provide working examples of C programs
* Help students practice problem-solving skills
* Serve as a reference for syntax and logic implementation

## 🧠 Topics Covered

The repository includes programs related to:

* Basic Syntax and Structure
* Input and Output (`printf`, `scanf`)
* Variables and Data Types
* Operators (Arithmetic, Relational, Logical)
* Conditional Statements (`if`, `else`, `switch`)
* Loops (`for`, `while`, `do-while`)
* Functions

  * With arguments and return value
  * With arguments and no return value
  * Without arguments and with return value
  * Without arguments and no return value
* Arrays
* Strings
* Basic Problem Solving

## 📂 Repository Structure

```
/basics
/conditionals
/loops
/functions
/arrays
/strings
/practice-problems
```

Each folder contains:

* Example codes
* Practice problems
* Sample outputs (when applicable)

## ▶️ How to Run

1. Compile the program:

   ```
   gcc filename.c -o output
   ```

2. Run the program:

   ```
   ./output
   ```

## 📝 Notes for Students

* Read the code carefully before running it.
* Try to modify the program and observe what changes.
* Practice writing the code from scratch instead of copying.
* Focus on understanding logic rather than memorizing syntax.

## ⚠️ Reminder

This repository is for learning purposes. Avoid copying solutions directly during exams or graded activities.

## 💡 Tip

The best way to learn programming is:

> **Practice, make mistakes, debug, and repeat.**
