# Student Information Display

## 🎯 Objective

Practice using `printf`, `scanf`, variables, and basic data types.

## 📋 Problem Description

Create a program that asks the user to input the following information:

* Full Name
* Age
* Height (in meters)

After receiving the input, display the entered information in a clean format.

## 📥 Input

* A string (Full Name)
* An integer (Age)
* A float (Height)

## 📤 Output

Display the user's information

## 🔍 Example

### Input

```id="in01"
Enter your full name: Juan Dela Cruz
Enter your age: 20
Enter your height (in meters): 1.75
```

### Output

```id="out01"
--- Student Information ---
Name   : Juan Dela Cruz
Age    : 20
Height : 1.75 meters
```

## 🧠 Hint

* Use `scanf` for numbers
* Use `fgets` for full name
