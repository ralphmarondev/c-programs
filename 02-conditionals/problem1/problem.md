# Grade Evaluator

## 🎯 Objective

Practice using `if`, `else if`, and `else` statements.

## 📋 Problem Description

Create a program that asks the user to input a student's grade (0–100).
The program should display the corresponding remark based on the following:

* 90–100 → Excellent
* 80–89 → Very Good
* 70–79 → Good
* 60–69 → Fair
* Below 60 → Failed

## 📥 Input

An integer (grade)

## 📤 Output

Display the remark

## 🔍 Example

### Input

```id="in02"
Enter your grade: 85
```

### Output

```id="out02"
Remark: Very Good
```

## 🧠 Hint

Use multiple `else if` conditions.
