# Simple Menu Calculator

## 🎯 Objective

Practice using `switch` statements.

## 📋 Problem Description

Create a program that displays a menu and asks the user to choose an operation:

1 - Addition
2 - Subtraction
3 - Multiplication
4 - Division

After choosing, the program should ask for two numbers and display the result.

## 📥 Input

* Choice (1–4)
* Two numbers

## 📤 Output

Display the result of the selected operation

## 🔍 Example

### Input

```id="in03"
Choose operation:
1 - Addition
2 - Subtraction
3 - Multiplication
4 - Division
Enter choice: 1

Enter first number: 10
Enter second number: 5
```

### Output

```id="out03"
Result = 15
```

## 🧠 Hint

Use `switch(choice)` and separate cases.
