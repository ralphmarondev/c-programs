# Multiplication Table

## 🎯 Objective

Practice using the `for` loop.

## 📋 Problem Description

Create a program that asks the user to input a number and displays its multiplication table from 1 to 10.

## 📥 Input

An integer

## 📤 Output

Multiplication table

## 🔍 Example

### Input

```id="in06"
Enter a number: 3
```

### Output

```id="out06"
3 x 1 = 3
3 x 2 = 6
3 x 3 = 9
...
3 x 10 = 30
```

## 🧠 Hint

Use a loop that runs 10 times.
