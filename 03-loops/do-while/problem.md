# Password Retry System

## 🎯 Objective

Practice using the `do-while` loop.

## 📋 Problem Description

Create a program that asks the user to enter a password.
The correct password is: **1234**

The program should keep asking the user until the correct password is entered.

## 📥 Input

An integer password

## 📤 Output

Display "Access Granted" once the correct password is entered

## 🔍 Example

### Input

```id="in05"
Enter password: 1111
Enter password: 2222
Enter password: 1234
```

### Output

```id="out05"
Access Granted
```

## 🧠 Hint

The loop must execute at least once.
