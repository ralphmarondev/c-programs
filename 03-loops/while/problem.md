# Number Counter

## 🎯 Objective

Practice using the `while` loop.

## 📋 Problem Description

Create a program that asks the user to input a positive integer **N**.
The program should display numbers from **1 up to N** using a `while` loop.

## 📥 Input

A positive integer N

## 📤 Output

Numbers from 1 to N

## 🔍 Example

### Input

```id="in04"
Enter a number: 5
```

### Output

```id="out04"
1 2 3 4 5
```

## 🧠 Hint

Initialize a counter before the loop.
